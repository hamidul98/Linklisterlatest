Smart link analysis plugin for WordPress that provides automatic link detection, analytics dashboard, and broken link detection.

== Description ==

LinkLister is a powerful WordPress plugin designed to help website owners manage, analyze, and optimize all the links on their website through a centralized, visually-rich admin dashboard.

= Key Features =

* **Automatic Link Detection**: Scans all pages and posts on your website and identifies every internal and external link
* **Link Analytics Dashboard**: Displays comprehensive stats including total posts, total links, internal vs external links, and 30-day sync activity
* **Broken & Redirected Link Detection**: Highlights broken or redirected links to help you fix SEO-damaging issues quickly
* **CSV Export**: Easily export all detected links for reporting or backup purposes
* **API-Powered Sync**: Connects to LinkLister SaaS platform via API key for advanced link management
* **Modern Admin UI**: Features a sleek WordPress admin interface with gradient styling, sync progress indicators, and chart visualizations

= How It Works =

1. Install and activate the LinkLister plugin
2. Register for a LinkLister account at https://linklister.com
3. Get your unique API key from the LinkLister dashboard
4. Enter the API key in the plugin settings
5. Click "Sync Now" to begin scanning all site links
6. Monitor results in the dashboard and export data as needed

= Benefits =

* Get a complete overview of all links on your site
* Identify problematic links affecting SEO performance
* Speed up SEO audits and link optimization
* Seamlessly integrate with external platforms for advanced insights
* Professional dashboard with real-time sync status

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/linklister` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to LinkLister in your WordPress admin menu
4. Navigate to Settings and enter your API key from https://linklister.com
5. Click "Sync Now" to start analyzing your site's links

== Frequently Asked Questions ==

= Do I need a LinkLister account to use this plugin? =

Yes, you need to register for a free account at https://linklister.com to get your API key and access the full features.

= Does this work with page builders? =

Yes, LinkLister works with all major page builders including Elementor, Divi, Beaver Builder, and others.

= How often should I sync my links? =

We recommend syncing after making significant content changes or at least once a week for active sites.

= Is my data secure? =

Yes, all data is transmitted securely via HTTPS and your API key is stored safely in your WordPress database.
